# Earthquake Detector

## Components

- Arduino UNO
- Accelerometer MPU6050
- LED
- Connecting wires
- Breadboard

## Wiring

### MPU6050 to Arduino connections
- VCC to 5V
- GND to GND
- SCL to A5 (SCL pin)
- SDA to A4 (SDA pin)

### LED to Arduino
- Positive pin of buzzer/LED to pin 8
- Negative pin of buzzer to GND
